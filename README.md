This project was created as an exercise with react creating a portfolio.
	
## Technologies
--------------------
Project is created with:
* react: ^16.13.1,
* react-dom: ^16.13.1,
* react-scripts: ^3.4.1
* react bootstrap 
* react spring 
	
## Setup
--------------------
### install the following locally:
```
$ cd ../dir
$ npm init
$ npm install
```

## Testing
--------------------
### To test this project, run the following in the terminal:
```
$ npm start
```
This will bring up a page on the browser from localhost 3001
--------------------
Alternatively, you can find the app deployed on [GitHub Pages](https://printicelawton.github.io/react_portfolio/)
